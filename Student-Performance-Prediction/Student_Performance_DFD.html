All Data Stores — Summary

Complete reference of every data store used in the system with their flows

IDData StoreTechnologyData StoredFlows
DS1 🗄️ User Collection MongoDB email, password (hashed), role, timestamps ← Register→ Login verify
DS2 🗄️ Student Collection MongoDB name, branch, semester, marks, attendance, status, riskScore, prediction ↔ Full CRUD→ Prediction trigger
DS3 🗄️ Prediction Store MongoDB studentId, score, riskLevel, recommendations, generatedAt ← ML result stored→ Analytics read
DS4 🗄️ Session Store localStorage + JWT JWT token (7d), user object (role, email, name, phone, address, bio) ← Login / updateUser→ Axios requests, Profile pages
DS5 🗄️ App Settings localStorage theme prefs, notification prefs, security settings, loginAttempts ↔ Settings page R/W
DS6 🗄️ ML Model Store Python / File Model weights, feature scalers, prediction logic → ML inference engine
